如何修改 DNS？

Windows

  1. 打开“控制面板” > “网络和 Internet” > “网络和共享中心”。
  2. 点击当前连接的网络。
  3. 点击“属性”，双击“Internet 协议版本 4 (TCP/IPv4)”。
  4. 选择“使用下面的 DNS 服务器地址”，填入首选和备用 DNS。

macOS

  1. 打开“系统偏好设置” > “网络”。
  2. 选择当前网络，点击“高级”。
  3. 切换到“DNS”标签页。
  4. 点击“+”号添加 DNS 服务器地址。

iOS / Android

  1. 进入“设置” > “Wi-Fi”。
  2. 点击当前 Wi-Fi 旁边的“i”图标或箭头。
  3. 找到“配置 DNS”，选择“手动”。
  4. 添加服务器地址。
