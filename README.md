# 公共 DNS 导航 (Public DNS Navigation)

这是一个简洁、美观的静态网页，旨在为用户提供全球知名公共 DNS 服务的快速索引和导航。

## 🌟 项目简介

本项目汇集了国内外主流的免费公共 DNS 服务商，提供经过分类整理的 IPv4/IPv6 地址、DoH/DoT 加密 DNS 接口地址，以及简单的配置教程。网站设计采用深色模式（Dark Mode），适配移动端和桌面端，并支持一键复制 IP 地址。

## ✨ 功能特点

*   **精选收录**：收录 Cloudflare、Google、阿里 DNS、腾讯 DNSPod、百度、字节跳动等国内外主流服务商。
*   **信息全面**：提供 IPv4 首选/备选地址、IPv6 地址、DNS over HTTPS (DoH) 和 DNS over TLS (DoT) 地址。
*   **一键复制**：点击 IP 地址旁的图标即可快速复制到剪贴板。
*   **响应式设计**：完美适配手机、平板和电脑屏幕。
*   **极简轻量**：纯 HTML/CSS/JavaScript 编写，无外部框架依赖，加载速度快。

## 🛠️ 使用方法

本项目为纯静态网页，无需复杂的环境配置。

### 方式一：直接打开
1.  下载本项目代码。
2.  直接双击打开 `index.html` 文件即可在浏览器中预览。

### 方式二：本地服务运行 (推荐)
如果你想获得更好的开发体验，可以使用简单的 HTTP 服务器运行：

**Python:**
```bash
python -m http.server
```

**Node.js (需安装 http-server):**
```bash
npx http-server
```

## 📂 文件结构

*   `index.html`: 网站主文件，包含所有内容结构和数据。
*   `style.css`: 样式文件，定义了深色主题、卡片布局和动画效果。
*   `README.md`: 项目说明文档。

## 📝 修改与贡献

如果你想添加新的 DNS 服务商或修改现有信息，请编辑 `index.html` 文件中的 `<section class="dns-grid">` 部分。每个服务商对应一个 `<div class="dns-card">` 元素。
